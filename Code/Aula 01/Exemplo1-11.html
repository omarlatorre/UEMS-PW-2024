Glossário de alguns termos técnicos empregados na computação.

ANSI
American National Standard Institute - Instituto Nacional Americano de Padrões.
ASCII
American Standard Code for Information Interchange - Código Americano Padrão para Intercâmbio de Informação.
Bit
Abreviação de Binary Digit e representa numericamente a presença (valor 1) ou ausência (valor 0) de tensão elétrica nos circuitos eletrônicos dos computadores.
Boot
Processo de inicialização ou reinicialização do computador.
Byte
Agrupamento de 8 bits que representa a menor unidade de informação que pode ser manipulada pelo processador.
CPU
Central Unit Processing - Unidade Central de Processamento. Normalmente considerado o próprio microprocessador.
DOS
Disk Operating System - Sistema Operacional de Disco.
LAN
Local Area Network, conhecida popularmente como rede local.
MSB
Most Significant Byte (Byte mais significativo).
RAM
Random Access Memory - Memória de Acesso Aleatório.
ROM
Read Only Memory - Memória Somente de Leitura.
